# 🎲 Random User Stage

A simple starter project created shortly after bootcamp to explore fetching and displaying random user data.
It serves as a learning exercise for working with APIs, state management and component rendering.

---

## 🛠 Tech Stack

* React (with Hooks)
* JavaScript / TypeScript
* CSS Modules or plain CSS
* Fetch or Axios for HTTP requests

---

## 📦 Installation & Running Locally

```bash
git clone https://github.com/marcus-koch-fs-dev/random-user-stage.git
cd random-user-stage
npm install   # or pnpm install
npm run dev   # or npm start
```

Open your browser to `http://localhost:3000` (or the port configured in your app).

---

## ▶️ Features / Usage

* Fetch a list of random users from [Random User API](https://randomuser.me/)
* Display user cards (name avatar location)
* Handle loading state and errors
* Optionally filter or paginate results
